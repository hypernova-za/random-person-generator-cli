## Setup

1. **Install Node.js**
   â†’ [https://nodejs.org/en/download](https://nodejs.org/en/download)

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run the app**

   ```bash
   tsc && node ./dist/main.js
   ```

## Compile to Binary with Deno

1. **Install Deno**

   ```bash
   npm install -g deno
   ```

2. **Compile**

   ```bash
   deno compile --output rpg ./src/main.ts
   ```
